Tooling

two-web/kit

A minimal starter toolkit for scaffolding 2Web projects without the noise.

Install

Grab the kit globally for quick bootstrapping.

npm install -g two-web/kit

Create

Generate a new project with sensible defaults and a wireframe baseline.

2web new my-app

Develop

Run the dev server and iterate with hot reload.

2web serve

File Layout

A tiny, predictable structure to keep focus on features—not plumbing.

src/ ├── __layout.html ├── __style.css ├── index.html └── components/
